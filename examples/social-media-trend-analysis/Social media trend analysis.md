# Social Media Trend Analysis

This example demonstrates how to analyze social media trends and engagement patterns by monitoring specific keywords on Twitter, tracking tweet metrics, and identifying influential retweeters to understand content virality and audience engagement.

## Prerequisites

1. **Twitter Setup**
   > Refer the [Twitter setup guide](https://central.ballerina.io/ballerinax/twitter/latest#setup-guide) here.

2. For this example, create a `Config.toml` file with your credentials:

```toml
bearerToken = "<Your Bearer Token>"
```

## Run the Example

Execute the following command to run the example. The script will print its progress to the console.

```shell
bal run
```
